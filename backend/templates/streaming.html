External Sensor POST
Use this payload to push data from your device:
POST {{ request.url_root.rstrip('/') }}/api/sensor
Content-Type: application/json
{
  "device_id": "R1",
  "temperature": 62.5,
  "vibration": 0.8,
  "speed": 2.5,
  "battery": 88,
  "timestamp": "2025-01-01T12:00:00Z"
}
Interval expectation: every 5s (offline after 15s).
How to stream
  1. Run built-in simulator via the buttons above or CLI `python simulator/sensor_simulator.py`.
  2. Point external devices to the `/api/sensor` endpoint with your auth cookie.
  3. Monitor device freshness and alerts in the Dashboard and Devices pages.
